# Python para Análise de Dados

### Conceitos introdutórios

Geralmente os dados estão organizados em tabelas, no formato "tidy" (observações nas linhas e variáveis nas colunas).

As variáveis em estatística estão divididas em:

- Qualitativas: nominais ou ordinais;
- Quantitativas: discretas ou contínuas.

Um dos principais erros de analytics, tanto no mercado profissional quanto na academia é a **ponderação arbitrária**. A ponderação arbitrária é o ato de atribuir pesos ou valores a **variáveis qualitativas**, extraindo destes pesos médias, por exemplo. Não existe fundamentação técnica ou estatística para esta ação.

As **relações entre variáveis** muda conforme a natureza da variável. Para variáveis qualitativas chamamos de **associação**. Para variáveis quantitativas chamamos de **correlação**.

Em **Machine Learning** existem técnicas supervisionadas e não supervisionadas:

- **Técnicas Supervisionadas:** o modelo é treinado usando um conjunto de dados rotulados. Isso significa que os dados de entrada já vêm acompanhados da resposta correta (o "rótulo" ou label). O algoritmo aprende a relação entre as entradas e saídas, permitindo que, após o treinamento, ele consiga prever resultados para novos dados que nunca viu antes.
  
- **Técnicas Não Supervisionadas:** é aquela em que conseguimos identificar padrões no comportamento dos dados através de técnicas exploratórias. Neste sentido, é possível analisar correlações entre as variáveis, mas não é possível realizar inferências para novas observações adicionadas à amostra.
